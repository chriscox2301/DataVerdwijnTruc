Deze software is een product van projectgroep DataVerdwijnTruc.

Als je de dataset die gebruikt wordt in de software wil aanpassen, dan kan dat middels een daarvoor bestemde tekst editor.
Ook kun je je eigen dataset toevoegen via het admin menu

Door in te loggen in het admin menu, vul je simpelweg de naam "roodkapje" in voordat je het spel start.
Je komt dan in een menu terecht waar je het nieuwe csv bestand kan uploaden.
In hetzelfde menu kan je de resolutie en audio volume aanpassen.

Gebruik van deze software is alleen toegestaan bij demonstratieve doeleinden of lesopdrachten.
Wij geven geen toestemming om dit programma en de code hiervan te gebruiken voor eigen gebruik/doeleinden.
Misbruik wordt gemeld.
